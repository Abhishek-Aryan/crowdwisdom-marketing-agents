Marketing Manager Agent

Hermes AIAgent Web Search Obsidian

  • Identifies top 3 acquisition channels with cost estimates
  • Creates 5 messaging angles (fear, aspiration, social proof)
  • Builds a 30-day content calendar
  • Analyzes 4 competitors (Motley Fool, Seeking Alpha, YouTubers, Discord)

Ads Scraper Agent

Apify API meta-ad-library-scraper JSON

What It Does

  • Calls Apify actor gTebjMDkz25esWXsY (meta-ad-library-multi-search-scraper)
  • Searches for "stock trading signals" and "trading newsletter"
  • Collects 100 ads per keyword (200 total)

  • Selects top 10 by hook strength, value prop, social proof, pain specificity

Pain Point Extractor Agent

Hermes AIAgent Eugene Schwartz Obsidian

What It Does

  • Reads selected_ads.json (top 10 winning Meta ads)
  • Extracts 5 deep emotional pain points (shame, knowledge gap, time poverty, betrayal, FOMO)
  • Identifies 15 specific fear triggers with direct ad quotes
  • Analyzes social proof patterns (numbers in 9/10 ads is dominant)
  • Builds swipe file of power words and emotional vocabulary
  • Extracts 10 reusable hook templates

Input → Output

Input
selected_ads.json (10 curated trading ads) · Eugene Schwartz "Breakthrough Advertising" methodology
Output
02_Pain_Point_Analysis.md (25.7 KB) — 5 pain points, 15 fear triggers, 10 hook templates, power words
2C

Ad Script Writer Agent

Hermes AIAgent PAS Framework Obsidian

What It Does

  • Reads pain analysis + YouTube research for context
  • Writes Script 1: "The Pain Opener" (30s, Problem-Agitate-Solution-CTA)
  • Writes Script 2: "Social Proof" (45s, Hook-Proof-Mechanism-Offer-CTA)
  • Writes Script 3: "Pattern Interrupt" (30s, Visual Hook-Counterintuitive-Bridge-CTA)
  • Labels every line: [HOOK] [PROBLEM] [AGITATE] [SOLUTION] [CTA]
  • Creates 10 alternative hooks for A/B testing

Input → Output

Input
02_Pain_Point_Analysis.md · 07_YouTube_Research.md · CrowdWisdomTrading product context
Output
03_Ad_Scripts.md (8.9 KB) — 3 video scripts + 10 hook variations + production notes
3

Influencer Outreach Agent

Hermes AIAgent Web Search Browser

What It Does

  • Searches for 10 trading influencers (200K+ followers) across YouTube, IG, TikTok, X
  • Collects: name, handle, followers, content style, audience level, contact info
  • Ranks by fit score (9/10 = Meet Kevin, Ricky Gutierrez, Trades By Tom)
  • Writes 5 personalized cold DMs (under 150 words each)
  • Creates follow-up template (1 week later) and positive-reply template

Input → Output

Input
Web search results · YouTube channel data · Instagram/TikTok/X profiles · CrowdWisdom product context
Output
04_Influencer_Research.md (11.6 KB) · 05_Influencer_Outreach.md (7.9 KB) — 10 profiles + 5 DMs + templates
4

Email Sequence Agent (Bonus)

Hermes AIAgent Copywriting Obsidian

What It Does

  • Writes 5-email welcome + nurture sequence (free → Pro conversion)
  • Email 1 (Day 0): Welcome + first value delivery
  • Email 2 (Day 1): Founder story — "I lost $8,000 following one guru"
  • Email 3 (Day 3): Product demo — NVDA consensus signal example
  • Email 4 (Day 5): Social proof — 3 mini case studies
  • Email 5 (Day 7): Urgency close — objection removal + CTA
